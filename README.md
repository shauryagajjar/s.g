# Shaurya — Portfolio (Budget Buddy)

A modern, responsive single-page portfolio for Shaurya. This site highlights student-focused projects such as **Budget Buddy**, **Crypto Basics**, and robotics/embedded demos.

## Overview
This site is built with semantic HTML + Tailwind CDN.  
It's lightweight, accessible, and easy to host anywhere.

## Files

- `assets/` — optional images
